Mammographic Mass Data Set

829 cases, 7 Columns

Column Description

  1. BIRADS: BI-RADS (Breast Imaging Reporting and Data System). Mammography assessment codes assigned by a radiologist. Ranges from 0 to 6.
  2. age: The age of the patient (integer)
  3. shape: mass shape (round/oval/lobular/irregular)
  4. margin: mass margin (circumscribed/microlobulated/obscured/ill-defined/spiculated)
  5. density: mass density (high/iso/low/fat-containing)
  6. severity: target field. 0=benign, 1=malignant.
